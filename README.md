# 🎮 CRUD de Jogos

## 📌 Sobre o projeto

Este projeto é um CRUD (Create, Read, Update, Delete) de jogos, onde é possível adicionar, visualizar, editar e remover jogos de uma lista.

Cada jogo contém informações como:

* Imagem
* Nome
* Nota
* Status (jogando, zerado, dropado)
* Considerações

---

* Upload de imagem com preview
